# Sitio web TN Colombia

**Descripción:**
- Proyecto estático: sitio web corporativo de TN Colombia. Consta de páginas HTML, hojas de estilo CSS, scripts JS y una carpeta `img/` para activos.

**Dependencias / Bibliotecas usadas (referenciadas en HTML):**
- Bootstrap (CSS/JS)
- jQuery (en `js/vendor`)
- Font Awesome (vía `kit.fontawesome.com` y `css/fontAwesome.css`)
- plugins.js y `hero-slider`, `owl-carousel`, `lightbox` (scripts/estilos locales)
